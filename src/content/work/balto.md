[Balto](https://balto.ai) builds AI-driven listening software for large call centers. Prior to this project, their core product, _Real-Time Guidance_, was a desktop application that delivered automated suggestions and encouragement during live customer calls. While effective, it was narrow in scope and constrained future growth.

Building on Balto’s machine learning foundation, I conceived and validated multiple user flows that resulted in a new product offering, _Real-Time Coaching_. The module introduced a native chat experience and an array of data-visualization tools, giving managers an at-a-glance view of team activity and productivity.
